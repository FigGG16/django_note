使用豆瓣源安装django1.9

```
pip  install  -i  https://pypi.doubanio.com/simple/  --trusted-host pypi.doubanio.com  django==1.9
```

安装app驱动

```
pip  install  -i  https://pypi.doubanio.com/simple/  --trusted-host pypi.doubanio.com  pymysql
```

安装以下

```
pip  install  -i  https://pypi.doubanio.com/simple/  --trusted-host pypi.doubanio.com  certifi
pip  install  -i  https://pypi.doubanio.com/simple/  --trusted-host pypi.doubanio.com chardet
pip  install  -i  https://pypi.doubanio.com/simple/  --trusted-host pypi.doubanio.com idna
```
